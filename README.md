# 1º Projeto de Iniciação Científica
Resumo: O trabalho utiliza técnicas de análise de sobrevivência para descrever o tempo entre o início do tratamento e o óbito por câncer de mama em 36.421 mulheres adultas brasileiras, a partir de dados do INCA/Integrador-RHC entre 2010 e 2019. As censuras são exclusivamente à direita, ocorrendo quando a paciente sai precocemente do estudo ou não apresenta o evento até o final do acompanhamento. A função de sobrevivência, definida como a probabilidade de sobreviver além de um determinado tempo, é estimada de forma não paramétrica pelo estimador de Kaplan-Meier, com apresentação de curvas gerais e estratificadas
por características das pacientes, da doença e do tratamento. A significância dos efeitos desses atributos na sobrevida é avaliada por meio do teste log-rank, que compara distribuições de sobrevivência entre grupos distintos.

Projeto de iniciação científica feito sob orientação da professora Núbia Karla (IME-UFF).
