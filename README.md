# 1º Projeto de Iniciação Científica
Neste repositórios estão, entre outras coisas, os scripts utilizados para realização de análise de sobrevivência de pacientes mulheres com câncer de mama que iniciaram seu 1º tratamento entre 2010 e 2019 e cujas informações estão no banco de dados integrador RHC do INCA.

Projeto de iniciação científica feito sob orientação da professora Núbia Karla (IME-UFF).

Link para obtenção dos dados: https://irhc.inca.gov.br/RHCNet/visualizaTabNetExterno.action
